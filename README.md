# Główna strona VFly

## Szybkie techdemo/żeby było na szybko

Wypociny LR i BS

## Odplanie

### Development

```cmd
git clone https://github.com/SKN-vFly/main-site.git
cd main-site
npm install
npm run dev
```

### Production

```cmd
git clone https://github.com/SKN-vFly/main-site.git
cd main-site
docker compose build
docker compose up
```

## Konwencje

- NPM

- Tailwind > style w CSS
